File Download Example

This example demonstrates browser-compatible file downloads using the Blob API, replacing the legacy VBScript FileSystemObject.

How It Works

The modern approach uses:

  • Blob API to create file content
  • URL.createObjectURL() to create a download link
  • <a download> attribute to trigger download

No server-side file system access needed!
