## مرور کلی

MCMI-II (پرسشنامه چندمحوری بالینی میلون - ویرایش دوم) یک ابزار جامع ارزیابی روان‌شناختی است که شامل 175 سؤال درست/نادرست برای اندازه‌گیری الگوهای شخصیتی بالینی و سندرم‌های روان‌شناختی طراحی شده است.

### ویژگی‌های کلیدی
- **تعداد کل سؤالات:** 175 آیتم
- **فرمت پاسخ:** درست/نادرست (انتخاب دوتایی)
- **تعداد کل مقیاس‌ها:** 26 مقیاس
- **روش نمره‌گذاری:** نمرات خام وزن‌دار ← نمرات نرخ پایه (BR) ← نمرات تعدیل‌شده
- **اختصاصی جنسیت:** جداول تبدیل و هنجارهای متفاوت برای مردان و زنان
- **محیط‌های بالینی:** تعدیلات بر اساس محیط درمانی متفاوت است

### هدف
این دستورالعمل راهنمای دقیق، کامل و جامع برای موارد زیر ارائه می‌دهد:
- محاسبه تمام نمرات خام و تعدیل‌شده
- تبدیل نمرات با استفاده از جداول مناسب
- اعمال تمام فرمول‌های تصحیح
- تفسیر بالینی نتایج
- پیاده‌سازی مجدد الگوریتم نمره‌گذاری

## ساختار آزمون

### فرمت سؤالات
هر یک از 175 سؤال نیاز به یک پاسخ دوتایی دارد:
- **FALSE (0)** - خیر - دکمه رادیویی سمت چپ
- **TRUE (1)** - بلی - دکمه رادیویی سمت راست

سؤالات با برچسب **R1 تا R175** در فرم ارزیابی مشخص شده‌اند.

### کدگذاری پاسخ
- وقتی دکمه رادیویی با شاخص **0** انتخاب شود = پاسخ FALSE (خیر)
- وقتی دکمه رادیویی با شاخص **1** انتخاب شود = پاسخ TRUE (بلی)

## مقیاس‌های بالینی

MCMI-II شامل 26 مقیاس بالینی است که در چهار دسته اصلی سازماندهی شده‌اند:

### دسته 1: شاخص‌های تعدیل‌کننده (4 مقیاس)
این مقیاس‌ها سبک پاسخ‌دهی و نگرش آزمون‌دهی را ارزیابی می‌کنند:

1. **V (اعتبار - Validity)** - تشخیص پاسخ‌دهی تصادفی یا بی‌دقت (4 آیتم)
2. **Y (مطلوبیت - Desirability)** - سنجش سوگیری خودارائه مثبت (حداکثر 23 آیتم)
3. **Z (تحقیر - Debasement)** - سنجش سوگیری خودارائه منفی (حداکثر 46 آیتم)
4. **X (افشاء - Disclosure)** - سطح کلی افشای اطلاعات (محاسبه از سایر مقیاس‌ها)

### دسته 2: الگوهای شخصیتی بالینی (10 مقیاس)
الگوهای پایه اختلالات شخصیت:

5. **مقیاس 1 (اسکیزوئید - Schizoid)** - جدایی اجتماعی، سردی عاطفی (حداکثر 35 آیتم)
6. **مقیاس 2 (اجتنابی - Avoidant)** - اضطراب اجتماعی، حساسیت بیش از حد به طرد شدن (حداکثر 41 آیتم)
7. **مقیاس 3 (وابسته - Dependent)** - تسلیم‌طلبی، نیاز به حمایت (حداکثر 29 آیتم)
8. **مقیاس 4 (هیستریونیک - Histrionic)** - توجه‌طلبی، رفتار نمایشی (حداکثر 24 آیتم)
9. **مقیاس 5 (خودشیفته - Narcissistic)** - خودبزرگ‌بینی، نیاز به تحسین (حداکثر 26 آیتم)
10. **مقیاس 6A (ضد اجتماعی - Antisocial)** - نقض قوانین، بی‌توجهی به دیگران (حداکثر 31 آیتم)
11. **مقیاس 6B (پرخاشگر/سادیستی - Aggressive)** - خصومت، پرخاشگری (حداکثر 31 آیتم)
12. **مقیاس 7 (وسواسی - Compulsive)** - کمال‌گرایی، سختگیری (حداکثر 24 آیتم)
13. **مقیاس 8A (منفعل-پرخاشگر - Passive-Aggressive)** - منفی‌گرایی، مقاومت منفعلانه (حداکثر 34 آیتم)
14. **مقیاس 8B (خودشکست - Self-Defeating)** - خودتخریبی، مازوخیسم (حداکثر 35 آیتم)

### دسته 3: آسیب‌شناسی شخصیت شدید (3 مقیاس)
اختلالات شخصیت شدیدتر:

15. **مقیاس S (اسکیزوتایپال - Schizotypal)** - تفکر عجیب، انزوای اجتماعی (حداکثر 37 آیتم)
16. **مقیاس C (مرزی - Borderline)** - بی‌ثباتی، روابط شدید (حداکثر 44 آیتم)
17. **مقیاس P (پارانوئید - Paranoid)** - بدبینی، بی‌اعتمادی (حداکثر 31 آیتم)

### دسته 4: سندرم‌های بالینی (9 مقیاس)

#### سندرم‌های متوسط (6 مقیاس)
18. **مقیاس A (اضطراب - Anxiety)** - علائم اضطراب عمومی (حداکثر 25 آیتم)
19. **مقیاس H (جسمانی‌سازی - Somatoform)** - شکایات جسمی/سوماتیک (حداکثر 29 آیتم)
20. **مقیاس N (شیدایی دوقطبی - Bipolar Manic)** - ویژگی‌های شیدایی/هایپومانیک (حداکثر 36 آیتم)
21. **مقیاس D (دیس‌تایمیا - Dysthymia)** - افسردگی خفیف مزمن (حداکثر 32 آیتم)
22. **مقیاس B (وابستگی الکل - Alcohol)** - سوء مصرف/وابستگی به الکل (حداکثر 26 آیتم)
23. **مقیاس T (وابستگی مواد - Drug)** - سوء مصرف/وابستگی به مواد (حداکثر 45 آیتم)

#### سندرم‌های شدید (3 مقیاس)
24. **مقیاس SS (اختلال فکری/اسکیزوفرنیا - Schizophrenia)** - علائم روان‌پریشی (حداکثر 47 آیتم)
25. **مقیاس CC (افسردگی اساسی - Major Depression)** - افسردگی شدید (حداکثر 31 آیتم)
26. **مقیاس PP (اختلال هذیانی - Delusional)** - تفکر هذیانی (حداکثر 22 آیتم)

## فرآیند نمره‌گذاری

فرآیند نمره‌گذاری MCMI-II شامل چندین مرحله متوالی برای تبدیل پاسخ‌های خام پرسشنامه به نمرات بالینی معنی‌دار نرخ پایه (BR) است.

### خط لوله کامل نمره‌گذاری

```
مرحله 1: محاسبه نمرات خام (مجموع وزن‌دار برای هر مقیاس)
    ↓
مرحله 2: تبدیل به نمرات نرخ پایه با استفاده از جداول جنسیتی
    ↓
مرحله 3: اعمال تصحیح X (افشاء)
    ↓
مرحله 4: اعمال تعدیل DA (اضطراب/افسردگی)
    ↓
مرحله 5: اعمال تعدیل DD (مطلوبیت/تحقیر)
    ↓
مرحله 6: اعمال تعدیل DC (انکار/شکایت)
    ↓
مرحله 7: اعمال تعدیل INP (محیط بستری)
    ↓
مرحله 8: محاسبه نمرات نهایی نرخ پایه
    ↓
مرحله 9: محاسبه نمره افشاء X
    ↓
مرحله 10: تولید گزارش
```

## محاسبه نمره خام

### اصول کلی

1. **نمره‌گذاری وزن‌دار:** هر سؤال وقتی TRUE پاسخ داده شود، 0، 1، 2 یا 3 امتیاز می‌دهد
2. **پاسخ‌های دوتایی:** فقط پاسخ‌های TRUE (انتخاب شده = شاخص 1) نمره‌گذاری می‌شوند
3. **آیتم‌های اختصاصی مقیاس:** هر مقیاس مجموعه خاص خود از سؤالات و وزن‌ها را دارد
4. **حداکثر نمرات:** هر مقیاس حداکثر نمرات خام جنسیتی دارد

### مقیاس اعتبار (V)

**هدف:** تشخیص پاسخ‌دهی تصادفی یا بی‌دقت

**آیتم‌ها (همه با وزن = 1):**
- R62 (TRUE)
- R90 (TRUE)
- R152 (TRUE)
- R169 (TRUE)

**حداکثر نمره خام:** 4

**محاسبه:**
```javascript
function v() {
  let sum = 0;
  if (k.r62[1].checked) sum += 1;
  if (k.r90[1].checked) sum += 1;
  if (k.r152[1].checked) sum += 1;
  if (k.r169[1].checked) sum += 1;
  return sum;
}
```

**تفسیر:** نمرات ≥ 2 نشان‌دهنده اعتبار مشکوک است

## تبدیل به نرخ پایه

پس از محاسبه نمرات خام، آنها باید با استفاده از جداول جستجوی اختصاصی جنسیت به نمرات نرخ پایه (BR) تبدیل شوند.

### درک نمرات نرخ پایه

**نمرات نرخ پایه (BR)** شیوع یک صفت در جمعیت بالینی را نشان می‌دهند:
- نمرات BR از 0 تا 115 (معمولاً) هستند
- BR = 60 یعنی صفت در 60% بیماران بالینی مشابه وجود دارد
- BR ≥ 75 نشان‌دهنده اهمیت بالینی است
- BR ≥ 85 نشان‌دهنده برجستگی بالا است

### فرآیند تبدیل

1. محاسبه نمره خام برای مقیاس
2. جستجوی نمره خام در جدول مناسب جنسیتی
3. بازیابی نمره BR متناظر

### مثال: جداول تبدیل مقیاس Y (مطلوبیت)

#### جدول مردان (تابع checkybr)
```javascript
function checkybr(rawScore) {
  const maleTable = {
    0: 0, 1: 5, 2: 10, 3: 15, 4: 20, 5: 25, 6: 30,
    7: 34, 8: 39, 9: 43, 10: 46, 11: 50, 12: 56,
    13: 62, 14: 67, 15: 72, 16: 75, 17: 78, 18: 82,
    19: 85, 20: 90, 21: 95, 22: 100
  };
  return maleTable[rawScore] || 0;
}
```

#### جدول زنان (تابع checkfybr)
```javascript
function checkfybr(rawScore) {

    13: 63, 14: 67, 15: 71, 16: 75, 17: 80, 18: 85,

  return femaleTable[rawScore] || 0;
}
```

## روش‌های تعدیل

پس از تبدیل BR، پنج تعدیل متوالی نمرات را اصلاح می‌کنند:

### تعدیل 1: تصحیح X (افشاء)

**هدف:** تعدیل بر اساس میزان کلی باز بودن/محافظه‌کاری در آزمون‌دهی

#### مرحله 1: محاسبه نمره خام X

```javascript
rawX = (w[6] + w[11]) * 1.5 +
       (w[3] + w[4] + w[5] + w[12]) * 1.6 +
       w[7] + w[8] + w[9] + w[10];
```

که در آن:
- w[3] = نمره خام مقیاس 1
- w[4] = نمره خام مقیاس 2
- w[5] = نمره خام مقیاس 3
- w[6] = نمره خام مقیاس 4
- w[7] = نمره خام مقیاس 5
- w[8] = نمره خام مقیاس 6A
- w[9] = نمره خام مقیاس 6B
- w[10] = نمره خام مقیاس 7
- w[11] = نمره خام مقیاس 8A
- w[12] = نمره خام مقیاس 8B

#### مرحله 2: اعمال قانون گرد کردن

```javascript
let rx = rawX - Math.floor(rawX); // دریافت بخش اعشاری
if (rx === 0.5) {
  rawX = rawX + 0.1; // اگر دقیقاً 0.5 است، 0.1 اضافه کنید
}
let roundedRawX = Math.round(rawX);
```

#### مرحله 3: تعیین مقادیر تصحیح

بر اساس X خام گرد شده، xcor و hxcor را پیدا کنید:

```javascript
function getXCorrection(rrawx) {
  if (rrawx >= 145 && rrawx <= 149) return { xcor: 11, hxcor: 5 };
  if (rrawx >= 150 && rrawx <= 159) return { xcor: 10, hxcor: 5 };
  if (rrawx >= 160 && rrawx <= 169) return { xcor: 9, hxcor: 4 };
  // ... ادامه برای همه محدوده‌ها
  if (rrawx >= 577 && rrawx <= 590) return { xcor: -12, hxcor: -6 };
  return null; // نامعتبر
}
```

#### مرحله 4: اعمال تصحیحات

**xcor** برای مقیاس‌ها اعمال می‌شود: 1، 2، 3، 4، 5، 6A، 6B، 7، 8A، 8B، A، H، N، D، B، T  
**hxcor** برای مقیاس‌ها اعمال می‌شود: S، C، P، SS، CC، PP  
**بدون تصحیح** برای مقیاس‌ها: Y، Z

### تعدیل 2: تعدیل DA (اضطراب/افسردگی)

**هدف:** تعدیل برای اضطراب و افسردگی بالا که بر سایر مقیاس‌ها تأثیر می‌گذارد

#### مرحله 1: محاسبه مقدار DA

```javascript
function daadjust() {
  let dcorrect = rawbr[19] + xcor; // مقیاس D پس از تصحیح X
  let acorrect = rawbr[16] + xcor; // مقیاس A پس از تصحیح X
  
  if (dcorrect >= 85) {
    if (acorrect < 85) {
      return dcorrect - 85;
    } else {
      return acorrect + dcorrect - 170;
    }
  } else {
    return 0;
  }
}
```

#### مرحله 2: اعمال DA بر اساس محیط بالینی

تعدیل DA بر اساس محیط درمانی (فیلد d1) متفاوت است:

```javascript
let setting = parseInt(k.d1.value);
let da, dac;

switch (setting) {

  case 4: // بدون درمان فعلی

    if (dac > 10) dac = 10;
    break;
    
  case 2: // بستری

    if (da > 25) da = 25;
    dac = dacontain;
    if (dac > 20) dac = 20;
    // تعدیلات اضافی INP برای CC، SS، PP
    afterinp[22] = afterinp[22] + 8;  // SS
    afterinp[23] = afterinp[23] + 10; // CC
    afterinp[24] = afterinp[24] + 4;  // PP
    break;
    
  case 3: // بستری جزئی
  case 5: // مشاوره

    if (da > 15) da = 15;
    dac = Math.floor(dacontain * 0.75);

}
```

### تعدیل 3: تعدیل DD (مطلوبیت/تحقیر)

**هدف:** تعدیل برای سبک آزمون‌دهی (جعلی خوب در مقابل جعلی بد)

#### مرحله 1: محاسبه مقدار DD

```javascript
function ddadjust() {
  let dd = (rawbr[1] - rawbr[2]) / 10; // (Y - Z) / 10
  
  // قانون گرد کردن خاص
  let decimal = Math.abs(dd - Math.round(dd));
  if (decimal === 0.5) {
    if (dd === Math.abs(dd)) { // مثبت
      dd = dd + 0.1;
    } else { // منفی
      dd = dd - 0.1;
    }
  }
  
  let rdd = Math.round(dd);
  
  // اعمال محدودیت‌ها
  if (rdd > 10) rdd = 10;
  if (rdd < -10) rdd = -10;
  
  return rdd;
}
```

#### مرحله 2: اعمال DD به مقیاس‌های خاص

DD به مقیاس‌های زیر اعمال می‌شود: S، C (پس از DA)، P، A، H، D

### تعدیل 4: تعدیل DC (انکار/شکایت)

**هدف:** تعدیل مقیاس‌های سندرم بالینی بر اساس الگوی شخصیت

#### مرحله 1: یافتن بالاترین و دومین بالاترین مقیاس‌های شخصیت

```javascript
function dcadjust() {
  let biggest = 0;
  let bigger = 0;
  let g, gp;
  
  // یافتن بالاترین در میان مقیاس‌های 3-12 (الگوهای شخصیتی)
  for (let j = 3; j <= 12; j++) {
    let score = (j === 4 || j === 12) ? dabr[j] : aftercor[j];
    if (biggest < score) {
      biggest = score;
      g = j;
    }
  }
  
  // یافتن دومین بالاترین
  for (let j = 3; j <= 12; j++) {
    if (j === g) continue;
    let score = (j === 4 || j === 12) ? dabr[j] : aftercor[j];
    if (bigger < score) {
      bigger = score;
      gp = j;
    }
  }
  
  return { g, gp };
}
```

#### مرحله 2: اعمال تعدیلات DC

**شرط 1:** اگر بالاترین مقیاس 4، 6A، 7 یا دومین بالاترین مقیاس 4 یا 7 باشد:

```javascript

  afterdccor[13] += 4;  // S

}
```

**شرط 2:** اگر بالاترین مقیاس 8B یا 2، یا دومین بالاترین مقیاس 2 باشد:

```javascript
if (g === 12 || g === 4 || gp === 4) {
  afterdccor[13] -= 2;  // S
```

### تعدیل 5: تعدیل INP (بستری)

**هدف:** تعدیل اضافی برای محیط‌های بستری/بستری جزئی

در طول مرحله تعدیل DA اعمال می‌شود (مراجعه کنید به مرحله 2 تعدیل DA بالا).

**برای محیط 2 (بستری):**

- CC: +10

**برای محیط 3 یا 5 (جزئی/مشاوره):**

## راهنمای تفسیر

### محدوده‌های نمره نرخ پایه

| نمره BR | تفسیر | معنی بالینی |
|---------|-------|-------------|
| 0-34 | بسیار پایین | صفت اساساً غایب است |
| 35-59 | پایین | صفت از نظر بالینی معنی‌دار نیست |
| 60-74 | حاضر | صفت حاضر است، نظارت شود |
| 75-84 | برجسته | صفت از نظر بالینی معنی‌دار |
| 85-115 | بسیار برجسته | بسیار معنی‌دار، تشخیص محتمل |

### تفسیر اعتبار

#### مقیاس V (اعتبار)
- **0-1:** پروتکل معتبر است
- **2+:** اعتبار مشکوک، پاسخ‌دهی تصادفی/بی‌دقت محتمل
- **3-4:** پروتکل احتمالاً نامعتبر

#### مقیاس Y (مطلوبیت)
- **BR 0-39:** خودارزیابی واقع‌بینانه
- **BR 40-74:** محدوده عادی
- **BR 75-84:** مدیریت برداشت مثبت تا حدی
- **BR 85+:** انکار معنی‌دار، مجموعه پاسخ جعلی-خوب

#### مقیاس Z (تحقیر)
- **BR 0-39:** اغراق در مشکلات نیست
- **BR 40-74:** محدوده عادی
- **BR 75-84:** بزرگنمایی مشکلات تا حدی
- **BR 85+:** اغراق معنی‌دار، جعلی-بد، فریاد برای کمک

#### نمره X (افشاء)
- **0-25:** بسیار محافظه‌کار، تدافعی
- **26-50:** افشای عادی
- **51-74:** باز، رک
- **75-89:** بسیار باز، اغراق خفیف احتمالی
- **90-100:** اغراق محتمل یا پروتکل نامعتبر

### مراحل تفسیر پروفایل

1. **اعتبارسنجی پروتکل:**
   - بررسی مقیاس V (باید < 2 باشد)
   - بررسی نمره X (باید 26-89 باشد)
   - بررسی تعادل Y و Z
   - تأیید پاسخ به همه سؤالات

2. **شناسایی مقیاس‌های بالا:**
   - لیست همه مقیاس‌ها با BR ≥ 75
   - توجه به مقیاس‌ها با BR ≥ 85 (اولویت بالا)

3. **تحلیل الگوهای شخصیتی:**
   - بررسی مقیاس‌های 1-8B
   - شناسایی الگوهای غالب
   - توجه به الگوهای متناقض

4. **بررسی آسیب‌شناسی شدید:**
   - بررسی مقیاس‌های S، C، P
   - بررسی ویژگی‌های شخصیت شدید

5. **ارزیابی سندرم‌های بالینی:**
   - بررسی مقیاس‌های A، H، N، D، B، T
   - شناسایی خوشه‌های علامت فعلی

6. **بررسی سندرم‌های شدید:**
   - بررسی مقیاس‌های SS، CC، PP
   - ارزیابی آسیب‌روانی شدید

7. **یکپارچه‌سازی یافته‌ها:**
   - ترکیب در تصویر بالینی منسجم
   - در نظر گرفتن تعاملات بین مقیاس‌ها
   - ارتباط با سؤال ارجاع و ارائه بالینی

```javascript
function validateProtocol(responses, scores) {
  let issues = [];

  if (responses.filter(r => r === null).length > 0) {

  if (scores.xScore > 89) {
    issues.push("نمره X > 89: اغراق محتمل");
  }
  if (scores.xScore < 10) {
    issues.push("نمره X < 10: محافظه‌کاری شدید");
  }
  
  // X خام در محدوده معتبر
  if (scores.rawX < 145 || scores.rawX > 590) {
    issues.push("X خام خارج از محدوده معتبر");

    issues.push("هم Y و هم Z بالا: پاسخ‌دهی متناقض");
  }
  
  return {
    valid: issues.length === 0,
    issues: issues
  };
}
```
